MOZiE, el imprescindible

Esta herramienta nos va a solucionar mucho las cosas a los maquetadores: MOZiE. Básicamente, MOZiE es una aplicación gratuita para entornos Windows que nos permite comparar la renderización de una página web en Mozilla e Internet Explorer simultáneamente. Aunque nos ofrece otras funcionalidades, no menos útiles, como por ejemplo la emulación de resoluciones de pantalla.
